
# Student Result Management System

Beginner-friendly Python + SQL project demonstrating CRUD operations, database creation, and student result management.

## Features
- Create student and marks tables
- Insert, fetch, update student details
- Beginner-level Python script with SQLite
- Perfect for Cognizant GenC submission

## How to Run
1. Install Python 3
2. Run: `python3 student_results.py`
3. A database `students.db` will be created automatically
